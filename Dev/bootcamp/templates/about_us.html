Panda Hardware

Is a prototype of a tiny e-commerce website which has been built with Django/Python.

Panda Hardware main idea has started as a result of the first 4 modules of a Django Bootcamp and then this project had been extended to something bigger.

Within building apps, from time to time I was interested in adding other website tools that had led to additional raws of code and more pages.

Being a big fan of motorcycles & of coourse pandas, I decided to put bikes as a product line.
